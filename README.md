# Mortal-Kombat-Numero-Magico
Reglas del juego
El juego consiste en adivinar un número secreto entre 1 y 100 que Sub-Zero ha elegido. El jugador controla a Scorpion y tiene un máximo de 10 intentos para descubrirlo.
Después de cada intento fallido, el juego indica si el número secreto es mayor o menor que el ingresado. La barra de vida de Sub-Zero funciona como termómetro de proximidad: cuando está llena y azul significa que estás lejos (frío), cuando baja a la mitad estás relativamente cerca (tibio), y cuando está casi vacía y roja estás muy cerca (caliente). Los sonidos también cambian según la proximidad: una risa indica que estás lejos, el grito de Scorpion indica que te estás acercando, y "Finish Him" indica que estás a punto de adivinar.
La barra de vida de Scorpion representa los intentos restantes: baja un 10% con cada fallo. Si se agota antes de adivinar el número, Sub-Zero gana y aparece la pantalla de derrota. Si el jugador adivina el número antes de quedarse sin intentos, Scorpion ejecuta un FATALITY y gana la batalla.
Al terminar cada partida, ya sea victoria o derrota, el jugador puede iniciar una nueva batalla con el botón "Nueva Batalla".
